# 2D Python Shooter

A side-scrolling tactical shooter game built using Python and the Pygame library. This game features player movement, enemy AI, combat mechanics (shooting and grenades), tile-based level loading, and parallax scrolling backgrounds.

## üéÆ Features

* **Platforming Mechanics:** Running, jumping, and collision detection.
* **Combat System:**
    * Shooting mechanics with ammo management.
    * Grenade physics with explosive areas of effect.
* **Enemy AI:** Enemies patrol, detect the player within a vision range, and engage in combat.
* **Level System:** Tile-based map generation loaded via CSV files.
* **Pickups:** Collectible Health, Ammo, and Grenade boxes.
* **Visuals:** Parallax scrolling background and animated sprites (Idle, Run, Jump, Death).
